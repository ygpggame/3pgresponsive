レスポンシブデザインとは？

パソコン、スマホ、タブレット、どのデバイスで見るかによって画面サイズが変わるため、それぞれに合わせたデザインにする必要がある。

見ているデバイスの画面サイズに応じて、自動的に最適化したデザインを表示する手法。

メディアクエリとは？

特定の大きさの時のみにＣＳＳ適応することができるCSSの書き方。

PCファースト方式

PCを基準として、スマホやタブレットサイズのみを変える箇所を修正する。

モバイルファースト方式

スマホを基準として、PCやタブレットのみを変える箇所を修正する。

特定の画面以下のサイズの時のみCSSを適用する

例：800px以下の画面の時に contents クラスの幅を50pxにする

                    
    @media screen and (max-width: 800px) {
        .contents {
                width: 50px;
        }
    }

特定の画面以下のサイズの時のみCSSを適用する

例：800px以下の画面の時に contents クラスの幅を50pxにする

                    
    @media screen and (max-width : 600px) {
        .contents {
            width: 100px;
        }
    }
